# Node starter

## Install

```bash
npm install
npm run build
```

## Update dependencies
```
npm i -g npm-check-updates
ncu -u
npm install
```
## Run

```bash
npm run start
```
## Debug (auto reload and listen for debugger)

```bash
npm run dev
```

In intellij, create a debug configuration of type "Attach to nodejs/chrome" on localhost:9229
